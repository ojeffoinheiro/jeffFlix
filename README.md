## Clone this repository

```bash
$ git clone https://github.com/ojeffoinheiro/jeffFlix.git
```

## Install the dependencies

```bash
cd backend && npm i
cd mobile && npm i
```


## Running the Server 🌐

Enter on folder

```bash
$ cd backend
```

Run the server

```bash
$ yarn start
```

## Run on mobile :iphone:

Enter on folder

```bash
$ cd mobile
```

Run the application
```bash
$ react-native run-android
```
